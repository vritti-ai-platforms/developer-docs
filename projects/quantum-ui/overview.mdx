**@vritti/quantum-ui** is Vritti's internal component library - a modern, TypeScript-first React component library built on Radix UI primitives and styled with Tailwind CSS v4.

<Info>
**Package Details**
- Version: 0.2.7
- Repository: [github.com/vritti-hub/quantum-ui](https://github.com/vritti-hub/quantum-ui)
- Storybook: [vritti-hub.github.io/quantum-ui](https://vritti-hub.github.io/quantum-ui)
</Info>

## Installation

```bash
pnpm add @vritti/quantum-ui
```

### Peer Dependencies

Ensure you have these peer dependencies installed:

```bash
pnpm add react react-dom react-router-dom @tanstack/react-query
```

## Quick Start

### 1. Import CSS

Import the component library CSS in your app entry point:

```tsx
// main.tsx or App.tsx
import '@vritti/quantum-ui/index.css';
```

### 2. Configure quantum-ui

Create a configuration file for API integration:

```ts
// quantum-ui.config.ts
import { defineConfig } from '@vritti/quantum-ui';

export default defineConfig({
  axios: {
    baseURL: '/api',
    timeout: 30000,
    withCredentials: true,
  },
  auth: {
    tokenEndpoint: 'cloud-api/auth/token',
    refreshEndpoint: 'cloud-api/auth/refresh',
  },
  csrf: {
    endpoint: '/csrf/token',
    enabled: true,
  },
});
```

Apply configuration at app startup:

```tsx
// main.tsx
import config from './quantum-ui.config';
import { configureQuantumUI } from '@vritti/quantum-ui';

configureQuantumUI(config);
```

### 3. Use Components

Import components individually for tree-shaking:

```tsx
import { Button } from '@vritti/quantum-ui/Button';
import { TextField } from '@vritti/quantum-ui/TextField';
import { Card, CardHeader, CardTitle, CardContent } from '@vritti/quantum-ui/Card';

function LoginForm() {
  return (
    <Card>
      <CardHeader>
        <CardTitle>Login</CardTitle>
      </CardHeader>
      <CardContent>
        <TextField name="email" label="Email" type="email" />
        <TextField name="password" label="Password" type="password" />
        <Button type="submit">Sign In</Button>
      </CardContent>
    </Card>
  );
}
```

## Import Patterns

Two import patterns are available:

```tsx
// Individual imports (recommended - tree-shaking)
import { Button } from '@vritti/quantum-ui/Button';
import { TextField } from '@vritti/quantum-ui/TextField';

// Bulk imports (convenience)
import { Button, TextField, Card } from '@vritti/quantum-ui';
```

<Warning>
Always prefer individual imports in production applications to minimize bundle size.
</Warning>

## Project Structure

```

│   ├── components/               # 17 public components

│   ├── utils/                    # Utilities (formHelpers, axios)
│   ├── index.ts                  # Main barrel export
│   └── index.css                 # Theme tokens and styles
├── shadcn/                       # Base shadcn/ui components (internal)
```
