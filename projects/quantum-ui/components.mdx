quantum-ui provides 17 production-ready components built on Radix UI primitives.

## Button

Primary action component with loading state support.

```tsx
import { Button } from '@vritti/quantum-ui/Button';

// Basic usage
<Button>Click me</Button>

// With loading state
<Button loading>Processing...</Button>

// Variants
<Button variant="default">Primary</Button>
<Button variant="secondary">Secondary</Button>
<Button variant="destructive">Delete</Button>
<Button variant="outline">Outline</Button>
<Button variant="ghost">Ghost</Button>
<Button variant="link">Link</Button>

// Sizes
<Button size="sm">Small</Button>
<Button size="default">Default</Button>
<Button size="lg">Large</Button>
<Button size="icon"><Icon /></Button>
```

## TextField

Text input with label, description, and adornment support.

```tsx
import { TextField } from '@vritti/quantum-ui/TextField';

// Basic usage
<TextField
  name="email"
  label="Email Address"
  placeholder="you@example.com"
/>

// With description
<TextField
  name="username"
  label="Username"
  description="Must be 3-20 characters"
/>

// With error
<TextField
  name="email"
  label="Email"
  error="Invalid email format"
/>

// With adornments
<TextField
  name="price"
  label="Price"
  startAdornment={<span>$</span>}
  endAdornment={<span>.00</span>}
/>
```

## PhoneField

International phone number input with country selector.

```tsx
import { PhoneField, isValidPhoneNumber } from '@vritti/quantum-ui/PhoneField';

// Basic usage
<PhoneField
  name="phone"
  label="Phone Number"
  defaultCountry="US"
/>

// Validation
const isValid = isValidPhoneNumber('+14155551234');
```

## Sonner (Toast)

Toast notifications for feedback.

```tsx
import { Sonner, toast } from '@vritti/quantum-ui/Sonner';

// Add Toaster to your app root
<Sonner />

// Show toasts
toast.success('Changes saved');
toast.error('Something went wrong');
toast.info('New update available');
toast.warning('Session expiring soon');

// With description
toast.success('Account created', {
  description: 'Check your email to verify your account'
});

// With action
toast.error('Failed to save', {
  action: {
    label: 'Retry',
    onClick: () => saveData()
  }
});
```

## Field System

Low-level field primitives for building custom form layouts.

```tsx
import {
  Field,
  FieldSet,
  FieldGroup,
  FieldLabel,
  FieldContent,
  FieldDescription,
  FieldError
} from '@vritti/quantum-ui/Field';

<FieldSet>
  <FieldGroup>
    <Field>
      <FieldLabel>Email</FieldLabel>
      <FieldContent>
        <input type="email" />
      </FieldContent>
      <FieldDescription>Your work email address</FieldDescription>
      <FieldError>Invalid email format</FieldError>
    </Field>
  </FieldGroup>
</FieldSet>
```

### Field Orientations

```tsx
// Vertical (default)
<Field orientation="vertical">...</Field>

// Horizontal
<Field orientation="horizontal">...</Field>

// Responsive (vertical on mobile, horizontal on desktop)
<Field orientation="responsive">...</Field>
```

## Form Component

Smart form wrapper with react-hook-form integration. See [Form System](/projects/quantum-ui/form-system) for detailed documentation.

```tsx
import { Form } from '@vritti/quantum-ui/Form';
import { useForm } from 'react-hook-form';

const form = useForm();

<Form form={form} onSubmit={handleSubmit}>
  <TextField name="email" label="Email" />
  <PasswordField name="password" label="Password" />
  <Button type="submit">Sign In</Button>
</Form>
```
