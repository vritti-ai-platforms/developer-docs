Before setting up your development environment, ensure you have the following tools installed.

## Required Software

### Node.js 22+

We use Node.js 22 LTS for all projects.

<Tabs>
  <Tab title="macOS (Homebrew)">
    ```bash
    brew install node@22
    ```
  </Tab>
  <Tab title="nvm">
    ```bash
    nvm install 22
    nvm use 22
    ```
  </Tab>
  <Tab title="fnm">
    ```bash
    fnm install 22
    fnm use 22
    ```
  </Tab>
</Tabs>

Verify installation:
```bash
node --version  # Should be v22.x.x
```

### pnpm 9+

We use pnpm as our package manager for all projects.

```bash
# Install via corepack (recommended)
corepack enable
corepack prepare pnpm@latest --activate

# Or install directly
npm install -g pnpm
```

Verify installation:
```bash
pnpm --version  # Should be 9.x.x
```

### Docker & Docker Compose

Required for running PostgreSQL and other services locally.

<Tabs>
  <Tab title="macOS">
    Install [Docker Desktop for Mac](https://docs.docker.com/desktop/install/mac-install/)
  </Tab>
  <Tab title="Linux">
    ```bash
    # Install Docker
    curl -fsSL https://get.docker.com | sh

    # Add user to docker group
    sudo usermod -aG docker $USER
    ```
  </Tab>
  <Tab title="Windows">
    Install [Docker Desktop for Windows](https://docs.docker.com/desktop/install/windows-install/)
  </Tab>
</Tabs>

Verify installation:
```bash
docker --version         # Docker version 24.x+
docker compose version   # Docker Compose version v2.x+
```

### Git

Version control system.

```bash
# macOS
brew install git

# Verify
git --version  # git version 2.x+
```

## SSL Certificates (Local Development)

For local HTTPS development, you'll need mkcert:

```bash
# Install mkcert
brew install mkcert

# Install local CA
mkcert -install

# Generate certificates (done automatically by projects)
mkcert local.vrittiai.com localhost 127.0.0.1 ::1
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="Node.js version mismatch">
    If you see version errors, ensure you're using Node.js 22:
    ```bash
    # Check current version
    node --version

    # Switch version (nvm)
    nvm use 22

    # Switch version (fnm)
    fnm use 22
    ```
  </Accordion>

  <Accordion title="pnpm not found">
    Enable corepack first:
    ```bash
    corepack enable
    corepack prepare pnpm@latest --activate
    ```
  </Accordion>

  <Accordion title="Docker permission denied">
    Add your user to the docker group:
    ```bash
    sudo usermod -aG docker $USER
    # Log out and back in
    ```
  </Accordion>

  <Accordion title="Port already in use">
    Find and kill the process:
    ```bash
    # Find process on port
    lsof -i :3000

    # Kill process
    kill -9 <PID>
    ```
  </Accordion>
</AccordionGroup>
